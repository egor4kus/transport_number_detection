# frontend

Возможная структура (AI-generated):

```text
frontend/
  README.md

  index.html
  src/
    App.jsx
    api.js
```

Идея папки: здесь лежит веб-страница для демонстрации проекта. Она не должна содержать ML-логику, только загрузку файла, запрос к backend и отображение результата.

Минимальный пользовательский сценарий:

```text
1. Пользователь открывает страницу.
2. Загружает фото городского транспорта.
3. Нажимает кнопку распознавания.
4. Видит изображение с рамками.
5. Видит таблицу:
   - тип транспорта
   - номер маршрута
   - confidence
```

Для MVP достаточно:

```text
- загрузка изображения
- кнопка отправки
- индикатор загрузки
- отображение результата
- отрисовка bbox поверх изображения или рядом с ним
```
